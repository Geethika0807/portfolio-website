# Portfolio Website

A personal portfolio website built using HTML, CSS, and JavaScript.

## Features
- Responsive navigation bar
- About Me section
- Skills section
- Projects showcase
- Education section
- Contact section
- GitHub and LinkedIn links
- Smooth scrolling effect
- Modern UI styling

## Technologies Used
- HTML
- CSS
- JavaScript

## Projects Included
- Automatic File Organizer
- Mini Attendance System
- YouTube Automation Bot

## Purpose
This project was created to practice frontend web development and build a professional portfolio website.
